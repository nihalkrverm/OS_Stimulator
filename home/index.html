CPU AND PROCESSES

The computer's central processing unit (CPU) is the portion of a computer that retrieves and executes instructions. The CPU is essentially the brain of a CAD system. It consists of an arithmetic and logic unit (ALU), a control unit, and various registers.

MEMORY AND PAGING

Memory management is the process of controlling and coordinating computer memory, assigning portions called blocks to various running programs to optimize overall system performance. Memory management resides in hardware, in the OS (operating system), and in programs and applications.

Why are the Algorithms used by operating systems?

  • Increased Speed
  • Efficient Memory Usage
  • Access control and Memory Protection

A scheduling system allows one process to use the CPU while another waits for I/O, allowing the CPU to be fully utilised.

Efficient Usage of Memory

Using new methods we can allocate space whenever needed . We can de-allocate dynamic space whenever we are done with them.
