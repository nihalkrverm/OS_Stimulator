Disk Scheduling


The main purpose of disk scheduling algorithms is to select a disk request from the queue of I/O requests and decide the schedule when this request will be processed. There are three basic types of algorithms: FCFS, Scan and CScan.
The disk scheduling is done by Operating System and without any user intervention.
