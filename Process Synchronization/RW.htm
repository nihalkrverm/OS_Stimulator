Readers-Writers Problem


The readers-writers problem relates to an object such as a file that is shared between multiple processes. Some of these processes are readers i.e. they only want to read the data from the object and some of the processes are writers i.e. they want to write into the object. In this simulation of the problem, the program considers multiple readers who can read at a time and allows only one writer to modify the file. The readers and the writer cannot use the file simultaneously.
